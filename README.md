저만의 작은 일기장을 웹페이지로 구현해봤습니다.
main page와 요일 페이지로 구성되어 있습니다.

- Main Page
  
    중앙에 '오늘의 일기'라는 제 웹페이지명을 넣었고, 그 아래에 짧은 소개글을 추가했습니다.
    화면의 한자로 적힌 월,화,수,목,금,토,일을 클릭하면 해당하는 요일 페이지로 이동합니다.

- 요일페이지
  
    각 요일 페이지는 사진 한 장과 짧은 일기로 구성되어 있습니다.
    오늘은 ~요일 부분을 클릭하면 메인 페이지로 이동합니다.

    만약, 요일페이지에서 따로 공유하고 싶은 음식점이 있는 경우 음식 페이지로 이동하는 부분을 우측 하단에 추가해두었습니다.

- 음식 페이지

    음식 사진과 한 줄평, 가는 방법 그리고 제가 매긴 별점을 담은 페이지입니다.
    맛있게 먹은 것 부분을 클릭하면 메인 페이지로 이동합니다.
    가는 방법에는 해당 음식점의 네이버 지도를 불러왔습니다.
    별점 부분은 버튼을 클릭해야 별점이 보이도록 했습니다.
